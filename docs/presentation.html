Цель и задачи работы ✔
Цель: Проведение исследования и разработка программного решения — кроссплатформенного мобильного приложения для семантического поиска по локальной медиатеке с использованием встроенных LLM, обеспечивающего интеллектуальный поиск изображений при полной автономности и конфиденциальности данных.
Задачи:
  • Проанализировать предметную область и подходы к on-device инференсу
  • Исследовать существующие мобильные решения (системы-аналоги)
  • Сформировать функциональные и нефункциональные требования
  • Разработать архитектуру мобильного приложения (C4, Clean Architecture)
  • Реализовать систему индексации и семантического поиска (CLIP + HNSW)
  • Разработать пользовательский интерфейс (Jetpack Compose, Material 3)
  • Провести тестирование системы (unit, benchmark, функциональное)
  • Оценить производительность и экономическую эффективность
