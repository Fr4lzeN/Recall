offset = k × d × 4 байта

7
Математическое обеспечение

1. Генерация дескрипторов (CLIP)

fimg : ℝH×W×C → ℝd     ftext : tokens → ℝd

Изображения и текст проецируются в единое пространство Rd (d = 384 или 512)

2. Поиск ближайшего соседа

sim(q̂, v̂i) = q̂ · v̂i = Σ q̂j · v̂ij    |    Results = { i | sim ≥ τ }

Косинусное сходство нормированных векторов = скалярное произведение

3. Алгоритм HNSW (Hierarchical Navigable Small World)

Линейный перебор: O(n)   →   HNSW: O(log n)
Многоуровневый навигируемый граф.
Параметры: M=16, efConstruction=200, efSearch=50
